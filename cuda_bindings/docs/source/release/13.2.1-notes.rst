``cuda-bindings`` 13.2.1 Release notes
======================================

Bugfixes
--------

* Per-thread default stream mode would be used whenever the
  ``CUDA_PYTHON_CUDA_PER_THREAD_DEFAULT_STREAM`` environment variable was set,
  even if it was set to ``0``.
