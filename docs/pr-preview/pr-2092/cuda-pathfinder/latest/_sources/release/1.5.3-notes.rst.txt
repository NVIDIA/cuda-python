``cuda-pathfinder`` 1.5.3 Release notes
=======================================

Highlights
----------

* Add support for the Windows CTK 13.2.1 CUPTI DLL name
  ``cupti64_2026.1.1.dll`` so ``load_nvidia_dynamic_lib("cupti")`` can
  recognize the installed library on CTK 13.2.1 systems.
