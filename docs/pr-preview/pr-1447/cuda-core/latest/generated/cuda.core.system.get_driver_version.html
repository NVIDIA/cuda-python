cuda.core.system.get_driver_version#

cuda.core.system.get_driver_version(bool kernel_mode: bool = False) → tuple[int, int]#

Get the driver version.

Parameters:

kernel_mode (bool) – When True, return the kernel-mode driver version, e.g. 580.65.06. Otherwise, return the user-mode driver version, e.g. 13.0.1.

Returns:

version – Tuple in the format (MAJOR, MINOR).

Return type:

tuple[int, int]
