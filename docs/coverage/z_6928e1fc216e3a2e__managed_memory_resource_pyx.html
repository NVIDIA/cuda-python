39  

40 Parameters 

41 ---------- 

42 options : ManagedMemoryResourceOptions 

43 Memory resource creation options.

45 If set to `None`, the memory resource uses the driver's current

47 the driver's default memory pool is used. 

48

50 the memory resource. 

51

54 `False`), and closing the resource has no effect.
